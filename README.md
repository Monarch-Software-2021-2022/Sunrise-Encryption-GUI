# Sunrise-Encryption-GUI
For the TSA Project 2021-2022

## Current Functions

**Hashing Methods**
- MD5
- SHA1
- SHA256
- SHA384
- SHA512
- RIPEMD160

**Encryption Methods**
- OSSE
- Caesar Cipher

**Plugin Support**
- Load & Initalize Plugins
- Dedicated Info Files for Plugins
